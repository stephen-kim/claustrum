# Docs Structure

This repository keeps detailed documentation in `docs/content/` and publishes it via GitHub Pages (`apps/docs-site`).

Primary sources:
- `docs/content/en/architecture.md`
- `docs/content/en/home.md`
- `docs/content/en/installation.md`
- `docs/content/en/operations.md`
- `docs/content/en/security-and-mcp-io.md`
- `docs/content/en/release-notes.md`

Korean counterparts:
- `docs/content/ko/home.md`
- `docs/content/ko/installation.md`
- `docs/content/ko/operations.md`
- `docs/content/ko/security-and-mcp-io.md`
- `docs/content/ko/release-notes.md`

Docs site app:
- `apps/docs-site`
- `docs/meta/_sidebar.md` (hierarchical nav source)

Commands:
- `pnpm docs:dev`
- `pnpm docs:build`
